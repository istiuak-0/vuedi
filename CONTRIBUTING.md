# Contributing to VueDI

Thank you for your interest in contributing to VueDI!

## How to Contribute

1. **Reporting Issues**  
   - Open an issue if you find a bug or have a feature request.
   - Provide a clear description, steps to reproduce (if a bug), and expected behavior.

2. **Submitting Pull Requests**  
   - Fork the repository and create a branch for your changes.
   - Make small, focused commits with clear messages.
   - Ensure your code follows existing style and patterns.
   - Submit a pull request with a description of what you changed and why.

3. **Code of Conduct**  
   - Be respectful and constructive in all interactions.
   - Help maintain a positive and welcoming community.
